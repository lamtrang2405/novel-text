# AI Novel Template Generator

Transform a single creative prompt into multiple unique novel templates, full stories, and audio drama scripts using AI.

## Features

- **Novel Templates** — Generate multiple novel outlines from one creative brief (Gemini or DeepSeek)
- **Full Story** — Expand any template into a complete novel
- **Audio Drama Script** — Convert stories into script format with narration and dialogue
- **TTS & Scenes** — Generate narrator audio and scene images (Gemini TTS + image API)
- **Character Voices** — Female/male/narrator voice selection for audio

## Setup

1. Get API keys:
   - **Gemini**: [Google AI Studio](https://aistudio.google.com)
   - **DeepSeek**: [platform.deepseek.com](https://platform.deepseek.com)
2. Open the app and enter your key(s) in the API Configuration section
3. Click **Test** to verify connection, then generate!

## Tech

- Static HTML/CSS/JS — no build step
- Gemini API (templates, story, script, TTS)
- DeepSeek API (templates, story, script)
- Runs entirely in the browser
